public static function font(p_str:String, p_face:String = null, p_size:uint = 0, p_color:String = null):String

Use the HTML font method to format and display a string in a document.

Parameters

p_str:String — the string to be formatted.
 
p_face:String (default = null) — (optional) specifies the name of the font to use, you can specify a list of comma-delimited font names, in which case Flash Player selects the first available font.
 
p_size:uint (default = 0) — (optional) an Integer representing the size of the font.
 
p_color:String (default = null) — (optional) the string hexadecimal RGB triplet with the format #rrggbb.

Returns
String — the string formatted with the method.
