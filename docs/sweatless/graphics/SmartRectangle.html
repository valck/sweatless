Classpublic class SmartRectangle
InheritanceSmartRectangle sweatless.graphics.Graphic

The SmartRectangle is a simple and a very useful substitute to native property Graphics.drawRoundRectComplex this extends the internal class Graphic, with this you can set the line style, fill and gradient fill, or fill texture easily. Note the baseclass extends Shape not Sprite.

The SmartRectangle is a simple and a very useful substitute to native property Graphics.drawRoundRectComplex this extends the internal class Graphic, with this you can set the line style, fill and gradient fill, or fill texture easily. Note the baseclass extends Shape not Sprite.
